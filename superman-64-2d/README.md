# Superman 64 2D

A standalone browser game prototype that reimagines the ring-chasing chaos as a faster 2D arcade rescue mission.

## Run
1. Double-click `Play Superman 64 2D.bat`.
2. A terminal window will open for the game server.
3. Your browser should open the game automatically.

If that does not work, you can still serve `C:\GitHub\codex-sandbox` from any static file server and open `http://localhost:<port>/superman-64-2d/`.

## Controls
- Move: Arrow keys or `W/A/S/D`
- Boost: `Shift`
- Heat Vision: `Space`
- Pause: `P`
- Restart: `Restart` button

## Goal
Stabilize Metropolis by:
- rescuing 5 falling civilians
- destroying 6 drones
- using ring routes to restore city health and solar energy
